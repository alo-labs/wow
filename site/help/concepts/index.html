What is WOW?

WOW! is an AI-native autonomous agentic performance optimizer for WordPress. It is not a plugin — it is a WP-CLI package that runs a coordinated pipeline of specialist AI agents, each responsible for a specific optimization domain.

You trigger the entire pipeline with a single command:

/wow

From that point, WOW! audits your site, plans changes, takes a full backup, executes optimizations, verifies results with visual regression testing, and delivers a detailed report — all without further input.

The 5 Optimization Domains

WOW! organizes all optimization work into five domains. Each domain is handled by one or more specialist agents:

Domain Focus Key Metrics
Performance Core Web Vitals, page speed LCP, CLS, INP, TTFB
Security SSL configuration, HTTP headers, hardening SSL grade, header score
SEO Meta tags, structured data, sitemaps Crawlability, schema coverage
Caching Server, browser, and CDN cache rules Cache hit rate, TTFB
Image Optimization Compression, modern formats, lazy-load Image byte savings, LCP

The 11 Specialist Agents

WOW! deploys 11 agents. Each agent is scoped to a single responsibility, making the system auditable and safe to extend.

Agent Domain Responsibility
Orchestrator All Coordinates the full pipeline; sequences agents in dependency order
Audit All Runs Lighthouse, captures Core Web Vitals baseline, inventories plugins and theme
Backup Safety Creates a full site snapshot (database + files) before any change is made
Caching Caching Configures server-side, browser, and CDN cache headers and rules
Image Image Optimization Compresses images, converts to WebP/AVIF, adds lazy-loading attributes
Security Security Enforces SSL, sets security headers (CSP, HSTS, X-Frame-Options), and hardens WP config
SEO SEO Audits and repairs meta tags, Open Graph, JSON-LD schema, and XML sitemaps
CSS/JS Performance Minifies, defers, and tree-shakes CSS and JavaScript assets
Database Performance Optimizes slow queries, cleans up post revisions and transients
CDN Caching / Performance Integrates with CDN providers to push static assets to the edge
Testing Safety Visual regression detection — screenshots before/after every change

Any agent can be skipped individually with --skip-<agent> (e.g., /wow --skip-image).

Backup & Rollback

WOW! is designed to be safe by default. The Backup agent runs before any change is made and creates a full snapshot of your site:

  • Complete database export
  • Full wp-content/ file archive
  • Snapshot stored in wp-content/wow-backups/

If the Testing agent detects a visual regression after execution, WOW! automatically triggers a rollback to the snapshot taken in that run. No manual action is required.

You can also trigger a manual rollback at any time:

/wow --rollback

WOW! lists available restore points and prompts for confirmation before restoring.
